# dicom-anonymizer

CLI program written in Rust to anonymize DICOM files

```
anonymizer 0.1.0
Domenic Melcher

USAGE:
    cli [OPTIONS] --input <INPUT> --output <OUTPUT>

OPTIONS:
        --dry-run                                    
    -h, --help                                       Print help information
    -i, --input <INPUT>                              
    -o, --output <OUTPUT>                            
    -p, --patient-name <PATIENT_NAME>...             
        --patient-birth-date <PATIENT_BIRTH_DATE>    
        --patient-sex <PATIENT_SEX>                  
    -r, --remove-tags <REMOVE_TAGS>...               
    -V, --version                                    Print version information
```
